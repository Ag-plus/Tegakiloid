概要
Ag++

	VOCALOID3のコントロールパラメータをスマートフォンのタッチパネルを
	使って設定するプラグインです。
	コントロールパラメータを滑らかに書きたいけど、ペンタブレットを持ってない。
	けどスマートフォンなら持っているよという方のためのプラグインです。

Ag++++++
使用方法
Ag++++++

	(0)
	VOCALOID3をインストールしてあるパソコンとお手持ちのAndroid端末を同一ネットワークに接続して下さい。
	VOCALOID3をインストールしてあるパソコンはAndroid端末からのUDP信号を受信する必要がありますので、
	ファイアウォール等を設定している場合は、UDPの受信ポートを1つ開けておく必要があります。
	開けたポート番号はTegakiloidジョブプラグインの設定を行う際に必要になりますので忘れないようにして下さい。

	(1)
	Tegakiloid.apkをお手持ちのAndroid端末にインストールしてください。
	[設定]→[アプリケーション]→[提供元不明のアプリ]のチェックボックスをチェックしておいて下さい。
	Tegakiloid.apkをAndroid端末のSDカード上にコピーします。
	ファイルブラウザアプリ(アストロなど)でapkファイルをタップすると
	インストールが始まります。
	作業が面倒な場合はhttps://github.com/Ag-plus/Tegakiloid/blob/master/Android/Tegakiloid.apkでも同じものを配布していますので
	こちらからインストールして下さい。
	なお、https://github.com/Ag-plus/Tegakiloid/tree/master/iOSにiOS版の信号送信アプリのソースも置いてありますので
	iOSアプリ開発のできる環境をお持ちの方はiOSでもTEGAKILOIDを使用することができます。

	(2)
	VOCALOID3エディタ上でコントロールパラメータを操作したい範囲を選択し、
	TEGAKILOIDプラグインを実行します。
	ダイアログが表示されますので、スマートフォンの画面サイズ、信号待ち受けポート、
	操作するコントロールパラメータの種類を選択してOKを押して下さい。

	(3)
	Android端末でテガキロイド(試作Ver0.0.0.1)を起動します。
	Menuキーを押して[設定]を選択し、VOCALOID3をインストールしてあるパソコンのIPアドレスと
	信号待ち受けポート番号を入力し、戻るキーを押して起動画面に戻ります。
	Android端末のタッチパネル上でコントロールパラメータに設定したいグラフの形を指でなぞります。
	なぞり終わったらMenuキーを押して[終了信号送信]をタップします。

	(4)
	選択した範囲のコントロールパラメータのグラフがタッチパネル上で描いた線の形になります。
